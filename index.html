Don't Use ID's to Style Elements

Bad

#photo {
  background: #fff;
  box-shadow: 1px 3px 5px rgba(0,0,0,.3);
  padding: 10px;
}

Can't reuse this style since an ID can exist only once on a page.

Bad

#sidebar img {
  background: #fff;
  box-shadow: 1px 3px 5px rgba(0,0,0,.3);
  padding: 10px;
}

More overriding if you DON'T want an image with that style.

  height: 1px;

Pass Arguments to Mixins


@mixin flex-direction($direction) {
  -webkit-flex-direction: $direction;
  flex-direction: $direction;
}	


.row {
  display: flex;
  @include flex-direction(column);
}


.row {
  display: flex;
  -webkit-flex-direction: column;
  flex-direction: column;
}

Pass Content Blocks to Mixins


@mixin breakpoint($break, $type: min-width) {
  @if $type == min-width {
    @media (min-width: $break) { @content; }
  } @else {
    @media (#{$type}: $break) { @content; }
  }
}
